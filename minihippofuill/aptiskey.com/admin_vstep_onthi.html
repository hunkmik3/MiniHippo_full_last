Tạo học viên Ôn thi VSTEP
Tài khoản chỉ truy cập phân khu Ôn thi (bộ đề tự luyện). Không bị giới hạn theo lớp/buổi.
Nhập mã học viên và mật khẩu, hoặc để trống mật khẩu để hệ thống tự tạo.
Mặc định = hôm nay. Truy cập tự động hết sau 6 tháng.
Để trống để hệ thống tự tính, hoặc nhập để override.
Import học viên Ôn thi từ Excel / CSV
Cột hỗ trợ: accountCode, fullName, email, phone, band, startedOn, expiresAt, password. Định dạng .xlsx / .xls / .csv / .tsv / .txt. HV import ở đây sẽ thuộc module Ôn thi.
Hoặc paste dữ liệu trực tiếp (Tab / Comma separated)
Chọn file Excel/CSV hoặc paste dữ liệu để bắt đầu.
